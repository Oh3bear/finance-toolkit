功能增强 银企对账导出增强方案

当前导出结构

目前导出 Excel 包含两类 Sheet：

  • 汇总 — 整体统计数字（总账户数、完全对符、有未对符等）
  • 账户明细 — 每个账户一个 Sheet，包含匹配明细、M:N 组合、未对符项

新增：账户列表 Sheet（索引页）

在"汇总"之后、各账户明细之前，新增一个 "账户列表" Sheet，作为导航索引页。

导出增强要点

  • 新增"账户列表"Sheet 作为导航索引，放在第 2 个位置（汇总之后）
  • 账户列表包含：序号、账户号、笔数统计、匹配数、未对符数、状态标记
  • "跳转明细"列使用 Excel HYPERLINK 公式，点击直接跳转到对应账户 Sheet
  • Sheet 名称需做安全截断（>28 字符截断），hyperlink 引用使用截断后的名称
  • 状态列用绿色/琥珀色区分"完全对符"与"存在未对符"
  • 底部增加合计行，方便快速核对总笔数

视觉升级 UI 配色与视觉升级方案

设计目标

当前配色以翡翠绿/青绿色为主，虽然专业但视觉层次较单一。升级方向：

  • 蓝绿黄三色渐变 — 用蓝色增加科技感，黄色作为点缀提亮，绿色保留财务专业感
  • 少量像素风元素 — 在图标、分隔线、装饰元素中加入像素化几何形状
  • 排版整齐一致 — 侧边栏与主内容区对齐规则统一，卡片间距规范化

新色板定义

翡翠绿（主色保留）
#eaf3de → #1a9e75 → #0f3d2e
科技蓝（新增强调）
#e6f1fb → #378add → #042c53
琥珀黄（点缀提亮）
#faeeda → #ba7517 → #412402
中性灰（背景与文字）
#f8faf9 → #8a9a90 → #1a2e25

配色应用场景

元素 当前配色 新配色 说明
Primary（主按钮/强调） 翡翠绿 hsl(161,94%,30%) 保留翡翠绿 品牌主色不变，保持财务专业感
Accent（高亮/标签） 青绿色 hsl(174,60%,90%) 科技蓝 hsl(210,70%,55%) accent 改用蓝色，增加层次
成功/对符状态 绿色 保留绿色 对符状态保持绿色，符合直觉
警告/未对符状态 琥珀色 保留琥珀色 未对符保持琥珀警示
侧边栏渐变 绿→青绿单色渐变 蓝→绿→黄 三色渐变 增加视觉丰富度
头部渐变 绿→白→青绿 蓝→白→黄 与侧边栏呼应
M:N 合并标签 灰色 蓝色 合并处理用蓝色标记
快速对符标签 青绿色 蓝色 快速通道用蓝色区分

装饰元素 像素风元素设计

像素风不是要把整个 UI 做成像素游戏，而是在关键位置加入少量几何像素元素作为点缀：

像素菱形 状态指示点
像素六边形 Logo 变体
像素星形 快速对符标记
像素三角 方向箭头
像素十字 取消/删除

像素风应用场景

位置 当前 像素风改造
步骤指示器圆点 圆形 ● 像素菱形 ◆（带 1px border-radius 边缘锐化）
分隔线装饰 实线或虚线 像素虚线（4px 实线 + 4px 空白重复）
Logo 圆角矩形 SVG 保留当前，增加像素化边缘变体可选
统计卡片图标背景 纯色圆角背景 像素几何形状背景（菱形/六边形）
加载/进度条 圆角条形 像素块状进度（小方块拼接）

排版规范清单

  • 侧边栏：固定宽度 256px（展开）/ 64px（折叠），内部 padding 统一为 px-3 py-2
  • 主内容区：统一 max-width: 1200px，居中 mx-auto，左右 padding 统一为 px-6
  • 卡片间距：卡片之间统一 gap-4（16px），卡片内部 padding 统一 p-4（16px）
  • 统计卡片：统一 4~6 列 grid，圆角统一 rounded-xl（12px），阴影统一 shadow-sm
  • Tab 栏：统一用 flex-wrap + gap-1 + bg-muted p-1 rounded-lg 的容器样式
  • 表格：表头统一 bg-muted + sticky top-0，数据行 hover:bg-background
  • 按钮：主按钮用 primary 绿，次按钮用蓝色 accent，警告用琥珀黄
